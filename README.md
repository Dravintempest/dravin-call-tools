## 📦 Install Dulu

Copy dan paste perintah ini buat install semua yang dibutuhkan:

```
pkg update && pkg upgrade -y
```
```
pkg install nodejs git -y
```
```
git clone https://github.com/Dravintempest/dravin-call-tools
```
```
cd dravin-call-tools
```
```
npm install
```
```
npm start
```
